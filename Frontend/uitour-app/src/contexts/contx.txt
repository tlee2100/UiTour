context/

Chức năng: chứa React Context, dùng để chia sẻ dữ liệu giữa nhiều component mà không cần props đi xuống nhiều tầng.

Mục đích: lưu info chung, ví dụ User login, Theme.

Ví dụ:

// context/UserContext.jsx
import { createContext, useState } from "react";

export const UserContext = createContext();

export function UserProvider({ children }) {
  const [user, setUser] = useState(null);
  return (
    <UserContext.Provider value={{ user, setUser }}>
      {children}
    </UserContext.Provider>
  );
}

// App.jsx
<UserProvider>
  <AppRouter />
</UserProvider>

// Navbar.jsx
import { useContext } from "react";
import { UserContext } from "../context/UserContext";

const { user } = useContext(UserContext);


Navbar có thể dùng thông tin user không cần truyền props từ App.
