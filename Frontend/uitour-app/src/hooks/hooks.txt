hooks/

Chức năng: chứa Custom Hooks, là hàm React dùng lại cho logic, ví dụ quản lý auth, fetch API.

Mục đích: tách logic ra khỏi component, dễ tái sử dụng.

Ví dụ:

// hooks/useFetch.js
import { useState, useEffect } from "react";

export default function useFetch(url) {
  const [data, setData] = useState(null);
  useEffect(() => {
    fetch(url).then(res => res.json()).then(setData);
  }, [url]);
  return data;
}

// Home.jsx
import useFetch from "../hooks/useFetch";

const data = useFetch("/api/movies");


Không phải viết logic fetch lặp đi lặp lại trong từng component.
